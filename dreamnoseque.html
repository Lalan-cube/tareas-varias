¿Qué es Dreamweaver?

Dreamweaver es un software de Adobe que sirve para crear sitios web mediante la utilización de distintos lenguajes de programación y permitiendo ver el resultado de lo que se edita al instante. Este software de diseño web permite trabajar con JavaScript, HTML o CSS, entre otros lenguajes de programación.
¿Qué permite hacer Dreamweaver? Por ejemplo, diseñar el “Acerca de" de una página web, elaborar portafolios con distintas imágenes o crear los sitios web de un comercio electrónico, todo ello desde la versión de código, es decir, la para que “no se ve" de una página web. Además, los cambios que se realicen en el código se pueden ver de manera inmediata en la versión web sin tener un modo de edición aparte.
